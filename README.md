# git-portfolio-AndriiKovalenko

This project is a simple Java calculator made for the Git & Build Tools portfolio task.  
It is a console application built with Maven and tested with JUnit 5.

## About the project
The calculator can:
- add numbers
- subtract numbers
- multiply numbers
- divide numbers

It also:
- reads user input from the console
- checks invalid input
- stores calculation history
- exports history in JSON format

## Build tool
I used **Maven** for this project.

I chose Maven because it has a clear standard structure for Java projects, it is easy to manage dependencies, and it works well in IntelliJ IDEA.

## Dependencies
This project uses:
- JUnit 5
- Gson
- Apache Commons Lang3
